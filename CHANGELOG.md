# Changelog

## [1.1.0] – 2025-10-05

### Added

- Scrolling indicators for dropdown menus (when no scrollbar is present)

- Updated (dev) dependencies
- Optimized scrollbar handling

### Fixed

- Improved scrolling lock when using overlays like the mobile menu
